JEU JOUABLE SEULEMENT SUR MANETTE XBOX.


CONTROLES --> 

-Pour le mouvement du vaisseau, utilisez le stick gauche.

-Pour la rotation du vaisseau, utilisez le stick droit.

-Pour le tir, utilisez la gachette RB.


Il s'agit d'un Shooter Arena type "Survival".

Les météorites sont aimantées par le vaisseau, il faut les éviter et également les détruire pour obtenir un maximum de point.
